# code_generate
基于Vue.js + Element UI开发的多功能代码生成器，支持通过SQL语句或字段配置快速生成：  多种数据库（MySQL/Oracle/SQL Server/达梦）的DDL语句  MyBatis Plus实体类  Mapper接口及XML文件  Service层接口与实现  Controller层REST API  支持一键导出ZIP压缩包
